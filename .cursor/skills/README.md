# Cursor Skills

This directory contains Cursor agent skills that can be used in your development workflow.

## What are Skills?

Skills are reusable prompts and instructions that help Cursor AI assist you with specific tasks. They can be referenced in your code or conversations with Cursor.

## Structure

Each skill should be a separate `.md` or `.mdc` file in this directory. The skill file should contain:
- A clear title
- Description of what the skill does
- Any specific instructions or examples

## Usage

To use a skill in Cursor, you can reference it through the Cursor interface or include it in your workspace configuration.

## Adding New Skills

To add a new skill:
1. Create a new `.md` file in this directory
2. Give it a descriptive name (e.g., `code-review.md`, `testing-strategy.md`)
3. Document the skill with clear instructions

## Example Skills

Skills can cover various topics:
- Code review guidelines
- Testing strategies
- Documentation standards
- Refactoring patterns
- Architecture decisions
- And more...
