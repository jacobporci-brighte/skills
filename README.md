# Skills

A collection of Cursor agent skills for development workflows.

## Overview

This repository serves as a collection of agent skills for [Cursor](https://cursor.com/docs/context/skills). Skills are reusable prompts and instructions that help Cursor AI assist you with specific development tasks.

## Structure

```
.cursor/
  skills/
    README.md          # Documentation for skills directory
    [skill-name].md    # Individual skill files (to be added)
```

## Usage

Skills in this repository can be used in Cursor by:
1. Cloning or referencing this repository in your workspace
2. Accessing skills through the Cursor interface
3. Using skills as context for AI-assisted development

## Adding Skills

Skills will be added to the `.cursor/skills/` directory as `.md` files. Each skill should:
- Have a descriptive filename
- Include clear instructions
- Provide examples where applicable

## Documentation

For more information about Cursor skills, visit the [Cursor documentation](https://cursor.com/docs/context/skills).

## Contributing

Skills can be added by creating new markdown files in the `.cursor/skills/` directory following the established patterns.
